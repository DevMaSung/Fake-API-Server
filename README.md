# Fake-API-Server

1. Git Clone https://github.com/DevMaSung/Fake-API-Server.git

2. Open Visual Studio

3. Solution Build

4. IIS Setup

5. Save Json(The json file will be saved to the D:\dummyAPI)

6. Call http://{yourdomain}/api/json/get/{jsonKeyName}
